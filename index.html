Colors

Sea Crystal Blue

#45a7ba

The primary color of the pallette is to be used for major theme elements. It should be used for H1 headers and other elements that need to be dominant in the visual hirarchy.

Parka Yellow

#fcba03

Parka Yellow is to be used as a secondary color for the purposes of highlighting or accenting features to provide veriety. Can be used for section backgrounds, buttons, etc.

Rock Cleft Gray

#666263

This dark gray will be the color used for the bulk of the text against white backgrounds. It can be used for content not meant to be highlighted such as a footer background or less important buttons.

Fonts

Bitter

Bitter is a serif font to be used for headings. It can also be used with text that is larger or not in a paragraph. It can be used with weights 300 for normal or 500 for bold, with italics as an option.

Regular 300:

Almost before we knew it, we had left the ground.

Regular 300 italic:

Almost before we knew it, we had left the ground.

Bold 500:

Almost before we knew it, we had left the ground.

Bold 500 italic:

Almost before we knew it, we had left the ground.

PT Sans

This sans serif font is to be used for text that will be in sentences or paragraphs. It is also to be used for text that is smaller. Available weights are 400 for normal and 700 for bold, with italics as an option for both.

Regular 400:

Almost before we knew it, we had left the ground.

Regular 400 italic:

Almost before we knew it, we had left the ground.

Bold 700:

Almost before we knew it, we had left the ground.

Bold 700 italic:

Almost before we knew it, we had left the ground.
