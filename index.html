Colors

Sea Crystal Blue

#45a7ba

The primary color of the pallette is to be used for major theme elements. It should be used for H1 headers and other elements that need to be dominant in the visual hirarchy.

Parka Yellow

#fcba03

Parka Yellow is to be used as a secondary color for the purposes of highlighting or accenting features to provide veriety. Can be used for section backgrounds, buttons, etc.

Rock Cleft Gray

#666263

This dark gray will be the color used for the bulk of the text against white backgrounds. It can be used for content not meant to be highlighted such as a footer background or less important buttons.
